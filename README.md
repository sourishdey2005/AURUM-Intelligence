# AURUM AI – Global Adaptive Multi-Factor Portfolio Intelligence Platform

## Project Overview
AURUM AI is an institutional-grade financial intelligence platform for multi-factor portfolio management, risk analytics, and AI-driven market regime forecasting.

## Architecture
- **Backend:** FastAPI (Python 3.11)
- **Database:** Supabase (PostgreSQL)
- **Cache:** Redis
- **Task Queue:** Celery
- **Frontend:** React 18 (TypeScript, Tailwind, D3, Three.js)
- **Infrastructure:** Docker Compose
